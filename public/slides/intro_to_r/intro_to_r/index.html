Loops

  • For loops
  • While loops
for(i in 1:10){
  some function(i)
}

while(i in x){
  some function(i)
}
i is the iterator, it has no real significance and can be named anything you like. It is recommended to name it based on what you are using it to loop through to make your code easier to follow.
